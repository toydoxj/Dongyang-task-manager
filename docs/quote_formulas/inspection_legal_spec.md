# 시특법 점검 견적 산식 (정기/정밀점검/정밀안전진단)

> 출처: 시설물의 안전 및 유지관리 실시 등에 관한 지침 (국토교통부고시 제2022-539호, 2022.9.28)
> 적용 범위: **건축물** (콘크리트구조, 상업용 기준)
> 단위: 인·일 (고급기술자 기준), 단가 = 한국엔지니어링협회 통계법 건설부분 노임단가

---

## 0. 기본 산식 (제55~58조)

```
대가 = 기본과업비 + 선택과업비

기본과업비 = 직접인건비 + 제경비 + 기술료 + 직접경비

direct_labor = INT(mh × 단가)             # 단가 = 310,884원/일 (고급기술자, 2026)
overhead     = INT(direct_labor × p%)      # p = 110~120% (default 110)
tech_fee     = INT((direct_labor + overhead) × q%)  # q = 20~40% (default 20)
direct_expense = Σ(별표 25 항목)
subtotal     = direct_labor + overhead + tech_fee + direct_expense
adjusted     = subtotal × 조정%
final        = adjusted - 절삭
```

## 4. 시설물 형태에 따른 적용 — 제61조

### 4-1. 기본시설물만

```
전체인원수 = 기본시설물 기준인원수
```

### 4-2. 인접시설물 있음

```
전체인원수 = 기본시설물 기준인원수 + Σ(인접시설물 기준인원수 × 0.7)
```

### 4-3. 군집시설물 (별표 27)

| 조건 | 산식 |
|---|---|
| 모든 동 1,000㎡ 미만 | 합계 면적의 기준인원수 × 1.0 |
| 모든 동 1,000㎡ 이상 | 최대 면적 기준인원수 × 1.0 + Σ(부속 동별 × 0.7) |
| 1,000㎡ 미만+이상 혼합 | 최대 면적 기준인원수 × 1.0 + Σ(부속 동별 × 0.7) |

## 5. 직접경비 — 별표 25 (제59조)

| 항목 | 산정 | 비고 |
|---|---|---|
| **여비·현장체재비** | 체재비(공무원 여비규정 제2호표) + 여비(철도 일반실 1회 왕복) | 인원: 정기 **2인** / 정밀점검 **8인** / 정밀안전진단 **10인** |
| **차량운행비** | 차량 손료 + 재료비 | 2,000cc 이하 / 외업 4인 이내 1대 (4 초과 시 4인당 +1대) / 손료계수 1,547×10⁻⁷ / 주연료 10ℓ/일 / 잡품 = 주연료 × 10% |
| **현지보조인부 노임** | 외업인원 × **40%** × 특별인부 시중 노임단가 | |
| **위험수당** | 현지직접인건비 × **10~20%** | 시설물별 작업위험도 |
| **기계·기구 손료** | 별표 22 전체 직접인건비 × % | 정기·제2종 5% / 정밀안전진단·제1종 **10%** |
| **보고서 인쇄비** | 종류별 쪽수 × 부수 | 정기 50쪽×10부 / 정밀점검 300쪽×10부 / 정밀안전진단 400쪽×20부 (각 + 전자저장매체 5부) |

## 6. 통합 산식 (건축물 정기/정밀/진단)

```python
# Step 1: base 인.일 (별표 22 + 보간)
base_total, base_outdoor = interpolate_base(area_m2, type)
base_indoor = base_total - base_outdoor

# Step 2: 보정 곱 (별표 23 + 제62조)
correction = (
    structure_factor   # 별표 23 (1) — 1.00 default
    × usage_factor      # 별표 23 (2) — 1.00 default
    × aging_factor      # 제62조 2 — 1.00 default
    × complexity_factor # 제62조 1 — 1.00 default
    × prev_report_factor # 제62조 3 — 1.00 default
)
mh_total = base_total × correction
mh_outdoor = base_outdoor × correction
mh_indoor = mh_total - mh_outdoor

# Step 3: 직접인건비
rate = 310_884  # 고급기술자 건설 (매년 갱신)
direct_labor = INT(mh_total × rate)

# Step 4: 직접경비 (별표 25)
travel = (2 if 정기 else 8 if 정밀점검 else 10) × 왕복단가
vehicle = ceil(mh_outdoor / 4) × (손료 + 연료 + 잡품)
helper  = mh_outdoor × 0.40 × 시중노임
risk    = direct_labor × 0.10  # 10~20% default 10
machine = direct_labor × (0.05 if 정기/정밀점검 else 0.10)
print_  = type별 쪽수 × 부수 × 단가
direct_expense = travel + vehicle + helper + risk + machine + print_

# Step 5: 제경비·기술료
overhead = INT(direct_labor × 1.10)  # 110% default
tech_fee = INT((direct_labor + overhead) × 0.20)  # 20% default

# Step 6: 합계·조정·절삭
subtotal = direct_labor + overhead + tech_fee + direct_expense
adjusted = subtotal × adjustment_pct / 100
final    = adjusted - (adjusted % truncate_unit)
```

## 7. 입력 필요 자료 정리

### 7-1. 사용자 입력 (frontend form)

| 필드 | 타입 | 예시 | 비고 |
|---|---|---|---|
| `area_m2` | float | 5192 | 연면적 |
| `inspection_type` | enum | "정기점검" / "정밀점검" / "정밀안전진단" | |
| `structure_form` | enum | "철근콘크리트" 등 7종 | 별표 23 (1) |
| `usage` | enum | "업무용" 등 5종 | 별표 23 (2) |
| `aging_years` | int | 29 | 경과년수 |
| `complexity` | enum | "단순" / "보통" / "복잡" | 제62조 1 |
| `prev_report` | enum | "미제공" / "CAD" / "보고서+CAD" | 제62조 3 |
| **시설물 형태** | enum | "기본" / "인접" / "군집(소)" / "군집(대)" / "군집(혼합)" | 제61조 |
| `adjacent_facilities` | list[area] | [3000, 5000] | 인접/군집 시 부속 면적 list |
| **위험수당 %** | float | 10~20 | default 10 |
| **기계·기구 % override** | float | 5 또는 10 | default 종류별 |
| **여비 1회 단가** | int | KTX 등 | 고정값 또는 사용자 입력 |
| `overhead_pct` | float | 110~120 | default 110 |
| `tech_fee_pct` | float | 20~40 | default 20 |
| `adjustment_pct` | float | 87 | 사장 조정% |
| `truncate_unit` | int | 100,000 등 | 절삭 단위 |

### 7-2. Backend 자동 lookup (table 데이터)

| 데이터 | 위치 | 비고 |
|---|---|---|
| 별표 22 base 인.일 | hardcoded list | 건축물만 (다른 시설물은 추후) |
| 별표 23 구조형식별 조정 | hardcoded dict | 7종 |
| 별표 23 용도별 조정 | hardcoded dict | 5종 |
| 별표 24 보간 공식 | function | 선형 |
| 경과년수 보정 | hardcoded list | 5단계 |
| 단가 (고급기술자 건설) | `ENGINEERING_RATES_BY_GRADE` | 매년 1월 갱신 |

### 7-3. 추가 확인 필요한 정보

| 항목 | 현재 상태 |
|---|---|
| **여비 1회 단가** | KTX 일반실 / 시외버스 등 — 운영자가 시점별 시세 입력? 또는 default 50,000원? |
| **시중 특별인부 노임** | 매년 갱신. 한국엔지니어링협회 단가? 또는 다른 시세? |
| **차량 손료계수 1,547×10⁻⁷** | 차량 가격 곱 — 사장 운영 차량의 가격 필요. 또는 평균 추정치? |
| **차량 주연료비 단가** | 휘발유 ℓ당 시세. 매월 변동. 운영자 입력 또는 고정? |
| **인쇄비 단가** | 쪽당 단가, 매년 갱신 필요 |

위 추가 입력값들은 backend 상수로 hardcode할지, 운영자 변경 가능 dict로 관리할지 결정 필요.
