# 동양구조 업무관리 — 사용 매뉴얼

> 버전 0.0.1 기준

## 1. 설치

1. 사내 공유 폴더 또는 GitHub Releases에서 `동양구조 업무관리 Setup 0.0.1.exe` 다운로드
2. 더블클릭 → 설치 위치 선택 (기본 권장) → 설치 진행
3. 첫 실행 시 Windows SmartScreen 경고가 뜨면 **추가 정보 → 실행** 선택
   (코드사이닝 인증서 미적용 — 향후 개선 예정)
4. 시작 메뉴 또는 바탕화면 단축키 **"동양구조 업무관리"** 로 실행

## 2. 첫 실행

### 2.1 관리자 계정 생성
- 처음 실행 시 백엔드에 사용자가 없으면 **초기 관리자 계정 생성** 화면 표시
- 이름·아이디·비밀번호·이메일 입력 → 관리자 권한으로 자동 가입

### 2.2 일반 사용자 가입
- 이후 사용자는 로그인 화면에서 **"가입 신청"** 클릭
- 관리자 승인 후 로그인 가능

## 3. 주요 화면

### 3.1 대시보드 (홈)
- 월별 매출/수금 콤보차트 (최근 12개월)
- 진행단계별 칸반 보드 (7컬럼)
- 팀 부하 히트맵 / 업무유형 매출 트리맵
- 현금흐름 예측 / 지출 구분 월간 추이
- 시작 전 TASK 적체 알림

### 3.2 프로젝트
- 검색·필터(단계/팀/완료) → 카드 클릭 → 상세

### 3.3 프로젝트 상세
- 헤더: 프로젝트명·발주처·담당팀·진행단계
- 라이프사이클 타임라인 (수주 → 계약기간 → 완료, TASK 마일스톤 표시)
- 진행률 게이지 (프로젝트% + 수금률)
- 현금흐름 (수금/지출 누적 영역 + 용역비 목표선)
- 지출 구분 도넛 (Top 5)
- 업무 TASK 칸반 (시작 전/진행 중/완료/보류, → 버튼으로 상태 변경)
- 우상단 "노션에서 열기" 클릭 시 원본 페이지로 이동

### 3.4 내 업무
- 본인이 담당자로 등록된 프로젝트만 표시
- Today TASK (마감 D-day 컬러 코딩)
- 마감 임박 프로젝트 D-day 정렬
- 본인 담당 프로젝트 카드 그리드

### 3.5 유틸 런처
- MIDAS Civil/Gen NX, 노션 워크스페이스, Drive, KDS 기준 등 외부 도구 카드

## 4. 자동 업데이트

- 새 버전 게시 시 앱 시작 시점에 알림 → "다운로드" → "재시작" 으로 설치
- 자동 업데이트는 관리자가 PAT(Personal Access Token) 주입한 빌드에서만 작동
- PAT 없는 빌드는 수동으로 새 설치파일 다운로드 후 재설치

## 5. 데이터 위치

| 항목 | 경로 |
|---|---|
| 사용자 DB (계정/세션) | `%LOCALAPPDATA%\동양구조 업무관리\data\app.db` |
| 백엔드 로그 | `%APPDATA%\동양구조 업무관리\backend.log` |
| 환경변수 (.env) | 설치 디렉토리 또는 `%APPDATA%\동양구조 업무관리\.env` |

## 6. 문제 해결

### 백엔드가 시작되지 않음
- 시작 시 "백엔드가 예기치 않게 종료되었습니다" 다이얼로그 → 백엔드 로그 확인:
  - `%APPDATA%\동양구조 업무관리\backend.log`
- 가장 흔한 원인: `.env` 누락 (NOTION_API_KEY) 또는 8000~9999 포트 모두 점유

### 노션 데이터가 안 보임
- 좌상단 메뉴 → "사용자 관리" 또는 노션 직접 접근으로 노션 통합 권한 확인
- 통합 ID: 노션 my-integrations 페이지에서 "동양구조 업무관리 앱"
- 4개 DB 모두 통합과 공유되어 있어야 함:
  1. `(주)동양구조 업무관리 - 프로젝트`
  2. `업무TASK`
  3. `프로젝트 수금 내역`
  4. `회사 지출내역`

### "내 업무" 페이지에 데이터가 없음
- 노션 담당자 옵션과 정확히 일치하는 이름으로 본인 프로필 등록 필요
- 우상단 "사용자명" → 프로필 수정에서 변경

## 7. 관리자 작업

### 7.1 사용자 승인
- 좌측 메뉴 "사용자 관리" → pending 상태 사용자 → "승인" 클릭

### 7.2 노션 DB ID 변경
- `.env` 파일의 `NOTION_DB_*` 값 수정 후 앱 재시작
